# 📦 Box Makers India Website

A responsive static website for **Box Makers India**, developed using **HTML, CSS, and JavaScript**. The website showcases the company's products, packaging solutions, company information, catalogs, and contact details.

## 🚀 Features

- Responsive homepage
- Product catalog
- Corrugated box categories
- About Us page
- Contact Us page
- Packaging materials page
- Navigation bar with dropdown menus
- Common footer across all pages

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/divyam5858/Box-Makers-India.git
```

2. Open the project folder.

3. Open `boxmakers.html` (or `index.html` if renamed) in your browser.
